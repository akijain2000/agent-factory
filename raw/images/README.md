# Image Sources

Architecture diagrams, framework comparison charts, and visual references used in wiki articles and slide decks.

## How to add images

1. Save the image to this directory with a descriptive kebab-case filename (e.g., `llm-kb-architecture.png`)
2. Reference in wiki articles: `![description](../../raw/images/filename.png)`
3. Reference in slides: `![description](../raw/images/filename.png)`
