# Agent Factory Course
## 23 Modules, 6 Blocks, ~12 Hours

Zero to production agent building

---

# Block 1: Foundations (Modules 01-04)

- **What Are Agents** — agents vs chatbots vs copilots
- **Architectures** — single, multi-agent, hierarchical
- **The Agent Loop** — plan-act-observe-reflect, ReAct
- **System Prompts** — persona + constraints + tool instructions

*Empirical: state machines are the #1 code quality driver (+5.0)*

---

# Block 2: Core Building Blocks (05-08)

- **Tool Design** — function calling, MCP, structured I/O
- **Memory** — short/long/episodic/semantic, context engineering
- **Planning** — CoT, ToT, task decomposition
- **Error Handling** — retries, fallbacks, circuit breakers

*Empirical: error taxonomy with retryable flags = +3.0*

---

# Block 3: Patterns & Anti-Patterns (09-12)

- **Design Patterns** — Ng's 4 patterns, ReAct, Reflexion
- **Multi-Agent** — supervisor, pipeline, swarm, debate
- **Anti-Patterns** — god agent, over-tooling, premature autonomy
- **State Management** — checkpointing, persistence, git-as-memory

*Empirical: stubs and happy-path-only tests are top anti-patterns*

---

# Block 4: Frameworks (13-16)

- **Framework Selection** — decision matrix
- **LangGraph** — state graphs, persistence, HITL
- **OpenAI Agents SDK** — handoffs, guardrails, tracing
- **Anthropic** — tool use, extended thinking, computer use

*Cross-cutting: observability, error contracts, and tests matter regardless of framework*

---

# Block 5: Production (17-20)

- **Evaluation & Testing** — behavioral tests, benchmarks
- **Safety & Guardrails** — sandboxing, prompt injection
- **Observability** — tracing, cost tracking, debugging
- **Deployment** — serverless vs containers, scaling

*Empirical: observability = largest single improvement (+9.0 from zero)*

---

# Block 6: Mastery (21-23)

- **Protocols** — MCP, A2A, tool marketplaces
- **Self-Improvement** — learning loops, harness engineering
- **Capstone** — build a production agent (3 tracks)

*All 12 updated modules include autoresearch empirical notes*
